# API-Comparador-Viajes

API REST en Node.js y TypeScript para gestionar usuarios, rutas de viaje y calcular la compatibilidad entre ambos mediante un algoritmo propio. Los datos se almacenan usando el gestor de base de datos NoSQL MongoDB.

Características
  CRUD de usuarios
  CRUD de rutas de viaje
  Algoritmo de matching que devuelve un score de 0 a 100 con desglose
  Arquitectura modular preparada para escalar
  Frontend simple en HTML/CSS para probar la API

Tecnologías:
  Node.js
  TypeScript
  Express
  ts-node-dev
  HTML, CSS y JavaScript para el frontend básico
